Содержит модели поведения, которые *могут* быть усвоены конкретными элементами
звёздной системы. Эти модели делают элементы уникальными, живыми.

Модели сгруппированы по группам элементам, т.к. чаще всего поведение,
декларируемое для одной группы, не будет работать для другой. Например,
модель "Увеличить рост" для горы, дерева и муравья будут отличаться.
